Как получить бесплатный вебхук (5 минут, без платных тарифов):
  1. В Битрикс24 откройте Разработчикам → Другое → Входящий вебхук (или через поиск: «вебхук»).
  2. Отметьте право доступа crm (CRM) и сохраните. Если нужен только просмотр — есть отдельные права на чтение сделок.
  3. Скопируйте ссылку вида https://ваш-портал.bitrix24.ru/rest/1/xxxxxxxxxxxxxxxx/ и вставьте её в поле выше.
  4. Нажмите «Подключить». После успешного подключения появится кнопка «Скопировать ссылку для руководителя» — она откроет дашборд уже с данными, без ручного ввода вебхука.
Важно про постоянную ссылку: Эта страница — обычный файл. Чтобы у неё был постоянный адрес (а не файл на вашем компьютере), разместите его на бесплатном хостинге (GitHub Pages, Cloudflare Pages, Netlify) — попросите об этом отдельно, дам пошаговую инструкцию. Если открывать файл прямо с компьютера (двойным щелчком), часть браузеров блокирует запросы к Битрикс24 — это самая частая причина, почему подключение «не работает».
