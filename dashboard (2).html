Как получить бесплатный вебхук (5 минут, без платных тарифов):
  1. В Битрикс24 откройте Разработчикам → Другое → Входящий вебхук (или через поиск: «вебхук»).
  2. Отметьте право доступа crm (CRM) и сохраните. Для разбивки по менеджерам добавьте ещё право user (Пользователи) — иначе покажем только ID сотрудника.
  3. Скопируйте ссылку вида https://ваш-портал.bitrix24.ru/rest/1/xxxxxxxxxxxxxxxx/ и вставьте её в поле выше.
  4. Нажмите «Подключить» — дашборд сам найдёт все воронки сделок и построит отчёт.
Важно про постоянную ссылку: Эта страница — обычный файл. Чтобы у неё был постоянный адрес, разместите его на бесплатном хостинге (GitHub Pages, Cloudflare Pages, Netlify). Если открывать файл прямо с компьютера (двойным щелчком), часть браузеров блокирует запросы к Битрикс24 — это самая частая причина, почему подключение «не работает».
